# Logic Check Skill for Claude Code

**Логик-аудит рассуждений** — скилл для [Claude Code](https://docs.anthropic.com/en/docs/claude-code), который проверяет аналитические документы на логическую непротиворечивость и выводит структурированный отчёт с формальной логической символикой.

## Что делает

Скилл выполняет 6-фазный аудит текста:

1. **Извлечение понятийного аппарата** — составляет карту терминов с содержанием и объёмом
2. **Проверка определений** — круги, соразмерность, однозначность
3. **Проверка классификаций** — несовместимость видов, одно основание, полнота
4. **Проверка логических законов** — тождество, непротиворечие, исключённое третье, достаточное основание
5. **Проверка аргументации** — корректность импликаций, паралогизмы, модальные ошибки
6. **Формирование отчёта** — структурированный вывод с формулами и рекомендациями

## Какие ошибки находит

| Ошибка | Формула | Пример |
|--------|---------|--------|
| Круг в определении | `Dfd ∈ Dfn` | «Эффективный процесс — процесс, организованный эффективно» |
| Подмена понятия | `A ≠ A'` | Термин «клиент» меняет объём по ходу текста |
| Противоречие | `A ∧ ¬A` | В §2 утверждается X, в §5 отрицается |
| Неполная классификация | `X₁+X₂+X₃ ≠ U` | Пропущены виды в типологии |
| Пересечение видов | `Xᵢ ∧ Xⱼ ≠ ∅` | Категории классификации перекрываются |
| Необоснованный вывод | `¬∃B: B → A` | Утверждение без достаточного основания |
| Модальная ошибка | `◇A → □B` | Из возможности выведена необходимость |
| Ложная дилемма | `A ⊻ B` при `∃C` | Предложены 2 варианта, когда их больше |

## Установка

Скопируйте `SKILL.md` в директорию скиллов Claude Code:

```bash
mkdir -p ~/.claude/skills/logic-check
cp SKILL.md ~/.claude/skills/logic-check/SKILL.md
```

## Использование

В Claude Code используйте любой из триггеров:

```
проверь логику этого текста: [текст]
```

```
/logic-check [путь к файлу]
```

```
найди противоречия в этом документе: [текст]
```

## Теоретическая база

Скилл основан на:
- Классическая формальная логика (Аристотель, Лейбниц, Фреге, Тарский)
- Четыре закона логики: тождества, непротиворечия, исключённого третьего, достаточного основания
- Теория понятий: содержание и объём, правила определений и делений
- Когнитивные модели: Система 1/2 Канемана, экологическая рациональность Гигеренцера
